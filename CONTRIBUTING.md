# Developing
- Run `npm install`
- `npm install -g vsce`
- `npm run build`
- Install the built `.vsix` file in VS Code
- Press F5 in VSCode to start debugging the extension

# Publishing
Run 
- `npm install -g vsce`
- `vsce package`
- A packaged with name `Ionic-0.0.1.vsix` will be created which can be installed
